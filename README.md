# Robotics Starter Project - Build while learning

This project is designed to help build a solid foundation in robotics by guiding you through the process of building a modular, simulation-first, ROS2-based autonomous mobile robot that can later extend into AI perception and multi-agent swarm systems.

# Overview

## Repo Structure
The repo is organized into the following key components:
- `docs/`: Design log, experiment logs, and phase notes.
- `ros2_ws/`: The ROS2 workspace containing all the packages for the robot's software stack, including simulation, control, and perception.
- `Architecture.md`: High-level system architecture and design decisions.
- `Roadmap.md`: Detailed multi-phase roadmap for development.
- `Progress.md`: Ongoing progress tracking and next steps.
